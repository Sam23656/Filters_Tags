Фильтр join

Фильтр используется для объединения значений из списка в одну строку с использованием заданного разделителя

Синтаксис

{% verbatim %} {{ value|join:" // " }} {% endverbatim %}

Пример без фильтра join:

{{ value }}

Пример с фильтром join:

{{ value|join:" // " }}

Фильтр rjust

Фильтр используется для выравнивания текста по правому краю, путем добавления заданного количества символов слева до указанной ширины.

Синтаксис

{% verbatim %} {{ value|rjust:25 }} {% endverbatim %}

Пример без фильтра rjust:

{{ "Lorem" }}

Пример с фильтром rjust:

            {{ "Lorem"|rjust:25 }}

Синтаксис

Пример без фильтра dictsortreversed:

    2. {% endfor %}

Пример с фильтром dictsortreversed:

    2. {% endfor %}

Тег templatetag

Тег используется для вывода специальных символов, таких как фигурные скобки {}, как часть результата шаблона, без их интерпретации как тегов или переменных DTL.

Синтаксис

{% verbatim %} {% templatetag openbrace %} {{ value }} {% templatetag closebrace %} {% endverbatim %}

Пример без тега templatetag:

{{ value }}

Пример с тегом templatetag:

{% templatetag openbrace %} {{ value }} {% templatetag closebrace %}

Тег resetcycle

Тег используется для сброса цикла, созданного с помощью тега cycle, к начальному состоянию.

Синтаксис

{% verbatim %} {% for elem in some_list %} {% cycle 'odd' 'even' %} {{elem}} {% resetcycle %} {% endfor %}{% endverbatim %}

Пример без тега resetcycle:

{% for elem in some_list %}

{% cycle 'odd' 'even' %} {{elem}}

{% endfor %}

Пример с тегом resetcycle:

{% for elem in some_list %}

{% cycle 'odd' 'even' %} {{elem}}

{% resetcycle %} {% endfor %}

Тег with

Тег используется для создания временных переменных в шаблонах. Он позволяет присвоить значению переменной другое имя и использовать его внутри текущего контекста шаблона.

Синтаксис

{% verbatim %}{% with total_price=price %} Total Price: ${{ total_price }} {% endwith %} {% endverbatim %}

Пример без тега with:

{{ price }}

Пример с тегом with:

{% with total_price=price %} Total Price: ${{ total_price }} {% endwith %}

Тег verbatim

Тег используется для отключения обработки шаблонизатором специфических символов, что означает, что содержимое между тегом verbatim будет рассматриваться как литеральный текст и не будет интерпретироваться как шаблонный код.

Синтаксис

{% templatetag openblock %} verbatim {% templatetag closeblock %} {% templatetag openvariable %} value {% templatetag closevariable %} {% templatetag openblock %} endverbatim {% templatetag closeblock %}

Пример без тега verbatim:

{{ value }}

Пример с тегом verbatim:

{% verbatim %} {{ value }} {% endverbatim %}

Синтаксис

Пример с тегом block:

Тег cycle

Тег который позволяет циклически переключаться между значениями внутри блока шаблона.

Синтаксис

{% verbatim %} {% for num in nums %}

{% cycle 'even' 'odd' %} {{num}}

{% endfor %} {% endverbatim %}

Пример без тега cycle:

{% for elem in some_list %}

{{ elem }}

{% endfor %}

Пример с тегом cycle:

{% for elem in some_list %}

{% cycle 'even' 'odd' %} {{ elem }}

{% endfor %}
