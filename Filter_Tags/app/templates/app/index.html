Фильтр join

Преобразует строку в заглавный регистр, заставляя слова начинаться с прописного символа, а остальные символы - со строчного.

Синтаксис

{% verbatim %} {{ value|join:" // " }} {% endverbatim %}

Пример без фильтра join:

{{ value }}

Пример с фильтром join:

{{ value|join:" // " }}

Фильтр dictsortreversed

Преобразует строку в заглавный регистр, заставляя слова начинаться с прописного символа, а остальные символы - со строчного.

Синтаксис

{% verbatim %} {{ foods|dictsortreversed:'category' }} {% endverbatim %}

Пример без фильтра dictsortreversed:

    {% for food in foods %}
  1. {{ food.name }}, {{ food.category }}
    2. {% endfor %}

Пример с фильтром dictsortreversed:

    {% for food in foods|dictsortreversed:'category' %}
  1. {{ food.name }}, {{ food.category }}
    2. {% endfor %}

Фильтр default_if_none

Преобразует строку в заглавный регистр, заставляя слова начинаться с прописного символа, а остальные символы - со строчного.

Синтаксис

Пример без фильтра default_if_none:

{{ None|default_if_none:"nothing" }}
