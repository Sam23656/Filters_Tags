Фильтр join

Фильтр используется для объединения значений из списка в одну строку с использованием заданного разделителя

Синтаксис

{% verbatim %} {{ value|join:" // " }} {% endverbatim %}

Пример без фильтра join:

{{ value }}

Пример с фильтром join:

{{ value|join:" // " }}

Фильтр truncatewords_html

Фильтр используется для обрезки строки до заданного количества слов

Синтаксис

{% verbatim %} {{ value|truncatewords_html:2 }} {% endverbatim %}

Пример без фильтра truncatewords_html:

{{ value }}

Пример с фильтром truncatewords_html:

{{ value|truncatewords_html:2 }}

Фильтр rjust

Фильтр используется для выравнивания текста по правому краю, путем добавления заданного количества символов слева до указанной ширины.

Синтаксис

{% verbatim %} {{ value|rjust:25 }} {% endverbatim %}

Пример без фильтра rjust:

{{ "Lorem" }}

Пример с фильтром rjust:

            {{ "Lorem"|rjust:25 }}

Фильтр dictsortreversed

Фильтр используется для сортировки словаря в обратном порядке на основе значений в словарях

Синтаксис

{% verbatim %} {{ foods|dictsortreversed:'category' }} {% endverbatim %}

Пример без фильтра dictsortreversed:

    {% for food in foods %}
  1. {{ food.name }}, {{ food.category }}
    2. {% endfor %}

Пример с фильтром dictsortreversed:

    {% for food in foods|dictsortreversed:'category' %}
  1. {{ food.name }}, {{ food.category }}
    2. {% endfor %}

Фильтр default_if_none

Фильтр используется для задания значения по умолчанию, которое будет отображаться в шаблоне, если переменная или выражение равны (None)

Синтаксис

{% verbatim %} {{ value|default_if_none:"nothing" }} {% endverbatim %}

Пример без фильтра default_if_none:

{{ None }}

Пример с фильтром default_if_none:

{{ None|default_if_none:"nothing" }}

Фильтр floatformat

Фильтр используется для форматирования чисел с плавающей запятой (float) в шаблонах Django. Этот фильтр позволяет округлить число до указанного количества знаков после запятой и отобразить его в виде строки.

Синтаксис

{% verbatim %} {{ float_value|floatformat }} {% endverbatim %}

Пример без фильтра floatformat:

{{ float_value }}

Пример с фильтром floatformat:

{{ float_value|floatformat }}

Фильтр timeuntil

Фильтр используется для вычисления временного интервала между текущей датой и временной меткой, представленной в виде строки или объекта datetime.

Синтаксис

{% verbatim %}{{ date_to|timeuntil:date}}{% endverbatim %}

Пример без фильтра timeuntil:

{{ date }}

Пример с фильтром timeuntil:

{{ date_to|timeuntil:date}}

Фильтр используется для форматирования строковых значений с использованием специальных форматных строк

Пример без фильтра stringformat:

Пример с фильтром stringformat:

{{ float_value|floatformat:"2" }}
